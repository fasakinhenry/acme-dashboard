## <a name="overview"> ğŸ‘€ Overview</a>

The Acme Dashboard SaaS is a web application built using Next.js and Tailwind CSS. It is designed to provide users with a comprehensive dashboard experience, featuring various components such as charts, tables, and user profiles. It is basically a dashboard that allows users to see various metrics and data visualizations about recent revenue and latest invoices. It also has a sidebar for navigation and Invoices page with pagination where users can create, view, search, edit, and delete invoices. It has features like skeletal loading and optimistic UI updates to enhance user experience.

The purpose of this project is to get the hang of Next.js App Router by building a functional dashboard application. It is a complete SAAS that has landing page, authentication, dashboard, invoices management, and more.

## <a name="features"> ğŸ”‹ Features</a>

- **Responsive Design**: The application is fully responsive and works well on various screen sizes.
- **Dashboard Components**: The dashboard includes various components such as charts, tables, and user profiles.
- **Invoices Management**: Users can create, view, search, edit, and delete invoices with pagination support.
- **Authentication**: User authentication is implemented using NextAuth.js.
- **Optimistic UI Updates**: The application provides a smooth user experience with optimistic UI updates by streaming components.
- **Skeletal Loading**: Enhanced user experience during data fetching.
- **Performance Optimization**: Built with performance in mind using Next.js features.
- **Search Functionality**: Users can search through invoices easily.
- **Search Engine Optimization (SEO)**: Basic SEO practices are implemented using Next.js metadata API.

## <a name="quick-start"> ğŸ¤¸ Quick Start</a>

Follow these steps to get the project up and running locally:

**Prerequisites**:

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (v14 or later)
- [pnpm](https://pnpm.io/) (preferred package manager)
- [PostgreSQL](https://www.postgresql.org/) (We use a cloud database like Neon) - optional
- [Git](https://git-scm.com/)

**Steps**:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/fasakinhenry/acme-dashboard.git
   cd acme-dashboard
   pnpm install
   pnpm dev
   ```
2. **Set up environment variables**:

    Rename the `.env.example` file to `.env` and fill in the required environment variables such as database connection string and NextAuth.js secret.

3. **Run the development server**:

    ```bash
    pnpm dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
